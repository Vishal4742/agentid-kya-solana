# Support — AgentID KYA Protocol

Need help? Here's how to get it.

---

## 🐛 Found a Bug?

1. Search [existing issues](https://github.com/Vishal4742/agentid-kya-solana/issues) first — your problem may already be reported.
2. If it's new, [open a bug report](https://github.com/Vishal4742/agentid-kya-solana/issues/new) and include:
   - What you were trying to do
   - What you expected to happen
   - What actually happened
   - Steps to reproduce (commands, screenshots, error messages)
   - Your environment: OS, Node.js version, Anchor version, Solana CLI version

---

## 💡 Have a Feature Request?

[Open a feature request](https://github.com/Vishal4742/agentid-kya-solana/issues/new) with:
- A clear description of the feature
- Why it would be useful
- Any relevant examples or prior art

---

## 🔒 Found a Security Vulnerability?

**Do NOT open a public issue for security vulnerabilities.**

Follow the responsible disclosure process in [`.github/SECURITY.md`](./.github/SECURITY.md):
- Email the maintainer directly, or
- Use GitHub's private vulnerability reporting feature

Critical vulnerabilities will be triaged within 48 hours.

---

## 📖 Documentation Issues?

If something in the docs is wrong, unclear, or missing:
- [Open an issue](https://github.com/Vishal4742/agentid-kya-solana/issues/new) with the label `documentation`
- Or submit a PR directly — doc fixes are always welcome

---

## 🤝 Want to Contribute?

See [CONTRIBUTING.md](./CONTRIBUTING.md) for how to set up the project locally and submit a PR.
