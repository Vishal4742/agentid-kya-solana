# Scripts Guide

## demo-devnet.sh

Demonstrates agent registration on Solana devnet.

```bash
bash scripts/demo-devnet.sh
```

Prerequisites: Solana CLI, funded devnet wallet, .env configured

## sync-idl.ps1

Syncs Anchor IDL between local and deployed program.

```powershell
.\scripts\sync-idl.ps1
```

Prerequisites: PowerShell 7+, Anchor CLI
